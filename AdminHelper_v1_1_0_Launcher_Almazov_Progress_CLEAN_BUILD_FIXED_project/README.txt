Admin Helper Launcher Edition v1.1.0

Это полностью рабочая лаунчер-версия Admin Helper.

Что перенесено из стабильной версии:
- открытие/закрытие подсказки по F9;
- кнопка открытия подсказки;
- полноэкранная подсказка;
- фикс курсора;
- проверка обновлений;
- сайт и Telegram;
- иконка EXE;
- безопасный выход;
- защита от второго запуска;
- лог ошибок adminhelper_error.log.

Что добавлено из прототипа лаунчера:
- боковое меню;
- разделы: Главная, Правила, Команды, Подсказка, Обновления, Настройки, О программе;
- стиль лаунчера в духе сайта;
- карточки, статусы и навигация.

Сборка:
1. Запустить build.bat.
2. Готовый файл:
   bin\Release\net8.0-windows\win-x64\publish\AdminHelper.exe

Важно:
Это новая версия v1.1.0 Launcher Edition.
Стабильную v1.0.0 можно оставить как резервную версию.


MINIMIZE FIX:
- исправлена ошибка при сворачивании/возвращении окна:
  Rectangle '{X=0,Y=0,Width=0,Height=0}' cannot have a width or height equal to 0;
- добавлена защита во все отрисовки с градиентом;
- добавлена защита для RoundedPanel и закруглённых прямоугольников.


AUTOUPDATE GITHUB:
- добавлено автообновление через GitHub Releases;
- при запуске лаунчер тихо проверяет latest release;
- если версия GitHub выше текущей, появляется окно обновления;
- кнопка «Скачать и установить» скачивает AdminHelper.exe, закрывает программу, заменяет файл и запускает новую версию;
- файл в релизе обязательно должен называться AdminHelper.exe;
- тег релиза должен быть формата v1.1.1 / v1.2.0 и быть выше текущей версии.

Для теста:
1. Собери эту версию v1.1.0.
2. Загрузи её в GitHub Releases как v1.1.0.
3. Собери тестовую версию v1.1.1 из второго архива.
4. Загрузи её в GitHub Releases как v1.1.1.
5. Запусти v1.1.0 — она должна предложить обновиться до v1.1.1.


GITHUB UPDATE JSON FIX:
- исправлено чтение ответа GitHub API;
- GitHub отдаёт поля tag_name, html_url, browser_download_url, а не TagName/HtmlUrl;
- теперь программа корректно видит релизы v1.1.1, v1.2.0 и выше;
- проверка берёт список выпусков и выбирает самую большую версию с активом AdminHelper.exe.

Для проверки:
1. В GitHub должен быть опубликован выпуск v1.1.1.
2. В Активы этого выпуска должен быть прикреплён файл AdminHelper.exe.
3. Запусти старую версию v1.1.0 из этого проекта.
4. Нажми «Проверить обновления» или подожди авто-проверку после запуска.
