Что внутри

  • Краткая подсказка по частым нарушениям
  • Основные наказания по регламенту
  • Разделы: аккаунт, чат, RP, памятка админу
  • Удобное открытие прямо поверх игры

  • Быстро ориентироваться в наказаниях

Особенности

  • Один файл для запуска

  • Простое управление через F9
  • Сделано специально для админ-состава

Как установить и запустить

1

Скачай файл

Нажми кнопку «Скачать последнюю версию» и скачай файл AdminHelper.exe.

2

Запусти от администратора

Нажми правой кнопкой по файлу и выбери «Запуск от имени администратора».

3

Используй F9

В игре нажми F9, чтобы открыть или закрыть подсказку администратора.

Важно: если подсказка не открывается в игре, запусти программу от имени администратора. Для обновлений используй кнопку «Проверить обновления» внутри программы.
